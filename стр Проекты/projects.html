  • Калькулятор
    Цель: Создание калькулятора, который должен:
    - уметь выполнять основные арифметические операции,
    - уметь работать с десятичными и отрицательными числами,
    - иметь кнопку для очистки экрана и кнопку Backspace, которая удаляет последнюю введенную цифру,
    - иметь функцию памяти, позволяющую сохранять и вызывать число.


  • Список дел
    Цель: Создание простого приложения для составления списка с функциями:
    - вывод списка дел из массива,
    - добавление новых пунктов в список дел,
    - добавление поля ввода,
    - добавление кнопки для удаления элементов из списка.
